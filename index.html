Deployment Guide
── Deploying via Remix IDE (no install needed) ──
1. Open remix.ethereum.org
2. Create new file → paste the FreeToMintNFT.sol source
3. Compile with Solidity 0.8.20
4. Deploy → select "Injected Provider - MetaMask"
5. Fill constructor params → Deploy
6. Paste contract address above
── Supported Networks ──
Ethereum · Polygon · BNB Chain · Arbitrum · Base · Avalanche · Optimism
+ any EVM-compatible testnet or mainnet
